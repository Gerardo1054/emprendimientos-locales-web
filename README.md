# Emprendimientos Locales Web

## Problemática

Muchos emprendimientos locales no cuentan con presencia digital, lo que limita su visibilidad, alcance a nuevos clientes y oportunidades de crecimiento.

## Objetivo

Desarrollar un sitio web que permita mostrar información de emprendimientos locales, facilitando el contacto, la ubicación y el registro de nuevos negocios.
